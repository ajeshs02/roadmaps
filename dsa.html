JavaScript-specific notes
Use Map & Set, not plain objects
Map.get() returns undefined for missing keys — always check. Objects coerce keys to strings which causes bugs with numeric keys.
Sort numbers properly
Default .sort() is lexicographic. Always pass a comparator: arr.sort((a, b) => a - b). Forgetting this causes subtle wrong-answer bugs.
Queue pattern without shift()
shift() is O(n). For BFS queues, use an array with a head pointer index and advance it — avoids reallocation on large inputs.
Recursion helpers & BFS template
Write a consistent inner dfs(node, ...) helper. For BFS, always: init queue with root, loop while queue non-empty, push children per iteration.
Avoid string concatenation in loops
Strings are immutable — repeated += in tight loops is O(n²). Push to an array and join once at the end: parts.join('').
What not to memorise
Skip segment trees, Fenwick trees, advanced graph algorithms, heavy DP optimisations, suffix arrays. Out of scope for mid-level generalist roles.
